# Git notes

## Check repository status

```bash
git status
```

## Add all changes

```bash
git add .
```

## Create commit

```bash
git commit -m "Commit message"
```

## Push to GitHub

```bash
git push
```

## Basic workflow

1. Make a small change
2. Test it
3. Check status
4. Add changes
5. Commit changes
6. Push changes
