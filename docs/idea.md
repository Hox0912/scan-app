# Project idea

Scan App is a mobile document scanner application.

The user takes a photo of a paper document. The app detects the document, crops it, fixes the perspective, applies a scanner-style black-and-white filter, and exports the result as a PDF.

## Why this project?

Most existing scanner apps include watermarks, ads, account requirements, or paid features.

This project should be simple, free, and useful.

## Core features

1. Take a photo
2. Detect paper edges
3. Crop document
4. Fix perspective
5. Apply scanner filter
6. Export as PDF

## Future features

- OCR/text recognition
- Search inside scanned documents
- Automatic file naming
- Multi-page PDF files
