# AI usage

AI will be used in two ways.

## 1. During development

AI can help with:

- Explaining code
- Debugging errors
- Writing documentation
- Suggesting Git commit messages
- Learning Android development

## 2. Inside the app

AI can be used for:

- OCR/text recognition
- Extracting text from scanned documents
- Suggesting document names
- Finding dates and amounts in receipts

## Important rule

The app should not depend on paid cloud APIs.

The core scanner functionality must work for free.
