# Scan App

Scan App is a free Android document scanner application without watermarks.

The goal of this project is to create an app that can scan paper documents using a phone camera, crop document edges, enhance the image, convert it to black and white, and export the result as a PDF file.

## Main goal

Create a free document scanner app without watermarks.

## Planned features

- Take a photo of a document
- Detect document edges
- Crop the document
- Fix perspective
- Apply black-and-white scanner filter
- Export scanned pages as PDF
- Support multiple pages in one PDF
- Add OCR/text recognition later

## Technologies

- Android
- Kotlin
- CameraX
- OpenCV
- ML Kit OCR
- Git
- GitHub

## Project rule

The core app must work without paid APIs or paid services.
